## 下载Shadowrocket

1.用户中心首页获取公共美区ID 并登入苹果商店App Store，下载安装Shadowrocket。

![](https://i.loli.net/2019/04/10/5cade67000c03.png)

{% hint style="danger" %}
国区没有这个APP，必须切换美区。（注意：国区有很多类似名字的盗版软件，小心下错！
{% endhint %}
